然后我在打印之前先赋值：Counter._i = 5;，得到了{ [Function: Counter] _i: 5 }。

且Counter.prototype._i = 6;可以改变this._i的值。

总结：JS创建一个function的时候，同时会为其创建一个prototype对象（原型对象）。而在function中的this，就是指向这个prototype对象的。
即 this = [Function].prototype。
