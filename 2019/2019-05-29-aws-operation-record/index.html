aws-operation-record

May 29, 2019

1、安装aws-cli，安装python-pip失败的话请尝试apt install update之后再试试:

2、配置用户：

  • 到https://console.aws.amazon.com/iam/home?#/users 中去创建一个带有AdministratorAccess权限的user。
  • 将包含密钥的CSV文件下`载到本地。
  • 通过aws configure命令配置好用户。

3、s3存储桶操作：

  • 创建buckets：aws s3 mb s3://learnjs.infomation.pub，s3://后面的部分为buckets的名字。
  • 上传文件aws s3 sync . s3://learnjs.infomation.pub --acl public-read --exclude ".git/*" --delete

4、sqs队列操作：

  • 列出所有队列：aws sqs list-queues
